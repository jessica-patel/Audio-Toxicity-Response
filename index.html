Verbal Toxicity Response for Online Games

Graduate Atelier Assignment 1 - Individual Thematic/Topic Explorations

By Jessica Patel

This is a prototype for a potential system that could be implemented in online games for the purpose of reducing in-game verbal harassment. It transcribes audio, and determines whether it is harmful according to the following criteria: identity attack, insult, obscene, severe toxicity, sexually explicit, threat, toxicity. This is done through the Text Toxicity Detection model by TensorFlow. If toxic language is detected, warnings appear on the screen as shown below.
